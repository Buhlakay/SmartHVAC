Prior to running display_events.py:
- Google Calendar API key is generated and the corresponding Google Calendar account is supplied with the desired events
- Make sure the credentials.json file is in the same directory as display_events.py

To run display_events.py:
- Change the date within main() to see if the time is contained within a user-entered event
- Type 'python3 display_events.py'

To exit the program:
- Press CTRL+C to exit the program

Results:
- The program will return and print true/false if the date is contained within a user event or not
